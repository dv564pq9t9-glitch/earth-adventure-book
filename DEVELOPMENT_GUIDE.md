# 地球冒険の書 (Earth Adventure Book) - 開発ガイド 🌍✨

このドキュメントでは、オゾネさんのNote紹介ページ「地球冒険の書」のデザインコンセプト、技術構成、および実装詳細について解説します。

## 1. コンセプトとデザインテーマ 🎨

### テーマ: 「冒険の書 (Adventure Log)」
ウェブサイト全体を一つの「RPG（ロールプレイングゲーム）の冒険の書」に見立て、訪問者がオゾネさんの人生という物語を読み進める「冒険者」となるような体験を設計しました。

*   **世界観**: ファンタジー、魔法、古代の地図
*   **キーワード**: 旅、猫、本質、物語、クエスト
*   **ターゲット**: 自分の人生を物語として楽しみたい人々

### デザイン要素
*   **羊皮紙 (Parchment) テクスチャ**: コンテンツの背景に古紙の質感を適用し、デジタル画面でありながらアナログな温かみと歴史を感じさせる演出。
*   **Cinzel Font**: ファンタジー映画やゲームでよく使われる、格調高いセリフ体フォントを見出しに使用。
*   **魔法のエフェクト**: ホバー時の発光、背景のパーティクル（光の粒子）など、動的な要素で「魔法」を表現。

## 2. 実装技術スタック 💻

特別なフレームワーク（ReactやVueなど）は使用せず、標準的なウェブ技術のみで構築することで、軽量かつカスタマイズしやすい構成にしています。

*   **HTML5**: セマンティックなマークアップ（記事を`<article>`タグで囲むなど）を行い、構造を明確化。
*   **CSS3 (Vanilla)**:
    *   Flexbox & Grid Layoutによるレスポンシブデザイン。
    *   CSS Variables（変数）を使用した色の一元管理。
    *   Transitions & Keyframe Animationsによる滑らかな動き。
    *   メディアクエリを使用したモバイル対応。
*   **JavaScript (Vanilla)**:
    *   動的なパーティクル生成（背景のキラキラ）。
    *   スクロール連動のアニメーション制御。

### サイトの更新方法
1.  手元のパソコンでコードを編集する。
2.  変更を保存し、以下のコマンドでGitHubに送信（プッシュ）する。
    ```bash
    git add .
    git commit -m "更新内容のメモ"
    git push origin main
    ```
3.  数分後、Vercelが自動的に変更を検知し、公開サイトを更新します。
